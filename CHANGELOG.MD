## Lost Prophecies 1.0 ##

Rebuilt entire list from the ground up on a 2.5.2 MO2 install with OpenMW Player version 2.1.5 alpha release.
All mods are updated to their latest version as of 2026-04-26 (except Bardcraft which remains in version 1.2 to avoid skill framework).
All unutilized mod archives from the downloads folder have been cleared, reducing the modlist install size from 152GB to approximately 115GB.
Many superceded mods were removed, putting us down from 186 mods to 167 mods.
